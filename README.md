ИНСТРУКЦИЯ

A) Установка и запуск.

	1 -  Клонируйте себе репозиторий командой : git clone url
    
	2 -  Установите зависимости командой : npm install
    
	3 -  Запустите сервер командой : npm start
        
    4 -  Зайдите на http://localhost:8080/
    
B) Как пользоваться.

	1 -  Что бы найти свой IP нажмите на соответствующую кнопку.
    
	2 -  Строго соблюдайте ввод IP в поле поиска.
    
	3 -  так же есть кнопка для скрытия истории поиска.
	
С) Минусы

	1 -  Нет горантии 100% работоспособности.
    
	2 -  Периодически история поиска обновляется.
    
	3 -  При не корректном вводе происходит ошибк
